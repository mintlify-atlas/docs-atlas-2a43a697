Welcome to the Raycast Script Commands Gallery! Explore hundreds of community-contributed scripts organized by category. Each script is ready to install and customize for your workflow.

## What are Script Commands?

Script Commands are simple scripts that extend Raycast with custom functionality. They can be written in any language that runs on macOS (Bash, Python, Ruby, Swift, JavaScript, etc.) and integrate seamlessly into your Raycast experience.

### Getting Started

1. **Browse** the categories below to find scripts that match your needs
2. **Install** scripts by copying them to your Script Commands directory
3. **Customize** scripts to fit your specific workflow
4. **Share** your own creations with the community

Learn more in our [Getting Started Guide](/quickstart).

## Community Contributions

All scripts in this gallery are contributed by the Raycast community. You can:

- **View source code** on [GitHub](https://github.com/raycast/script-commands)
- **Report issues** or suggest improvements
- **Contribute** your own scripts to help others
- **Request features** in the community

## Popular Scripts

Some of the most popular scripts include:

- **Battery Info** - Display detailed battery statistics in your menu bar
- **Generate UUID** - Quickly generate and copy UUIDs
- **Toggle Dark Mode** - Switch between light and dark appearance
- **Spotify Controls** - Control music playback without leaving your keyboard
- **Git Repository Info** - View Git status and branch information
- **Network Info** - Check your IP address and network details
